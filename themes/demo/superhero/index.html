Buttons

Button tags

Use the button classes on an <a>, <button>, or <input> element.

Link 

<a href="#" class="btn btn-default" role="button">Link</a>
<button class="btn btn-default" type="submit">Button</button>
<input class="btn btn-default" type="button" value="Input">
<input class="btn btn-default" type="submit" value="Submit">

Options

Use any of the available button classes to quickly create a styled button. 

<button type="button" class="btn btn-default">Default</button>
<button type="button" class="btn btn-primary">Primary</button>
<button type="button" class="btn btn-success">Success</button>
<button type="button" class="btn btn-info">Info</button>
<button type="button" class="btn btn-warning">Warning</button>
<button type="button" class="btn btn-danger">Danger</button>
<button class="btn btn-link" type="submit">Link</button>

Global Add-Ons

Switch

Use switches to visualize states based on checkboxes. 

<label class="switch">
	<input type="checkbox">
	<div class="slider">
</label>
<label class="switch">
	<input type="checkbox">
	<div class="slider slider-primary">
</label>

<label class="switch">
	<input type="checkbox">
	<div class="slider rounded">
</label>
<label class="switch">
	<input type="checkbox">
	<div class="slider slider-rounded slider-primary">
</label>
